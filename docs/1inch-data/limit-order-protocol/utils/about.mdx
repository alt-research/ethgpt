# About

`@1inch/limit-order-protocol-utils` is JavaScript and Typescript compatible library that solves everything for you.

Is solves:
- Order creation
- Predicate building via DSL
- Order signing
- Order filling
- Order cancelling
- Order predicate checking and simulation
- Remaining order info querying
- Order decoding
- Predicate decoding
- Nonce management
- And all mentioned above for **RFQ orders** as well

We taking care on it to make everything up to date and make you migration over Limit Order Protocol Versions as smooth as possible.

- [1inch/limit-order-protocol-utils on GitHub](https://github.com/1inch/limit-order-protocol-utils/)
- Issues & PRs welcome!
